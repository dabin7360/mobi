设置移动端滚动条

::-webkit-scrollbar{
  width:8px;
}
::-webkit-scrollbar-track{
  background-color:#FFF;
}
::-webkit-scrollbar-thumb{
  background-color:#b0b0b0
}

overflow:auto;
-webkit-overflow-scrolling:touch;

	rgba(255,255,255,0)
//屏蔽输入框怪异的内阴影
